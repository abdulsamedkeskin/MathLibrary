# MathLibrary
Math Library for Python

# Installation
<pre><code>pip install MathLibrary</code></pre>

# Example
        
        from MathLibrary.mathlibrary import MathLibrary
        
        MathLibrary.fact(5)
        
     
### Output : 
#### 120
