# MathLibrary
Math Library for Python

<h1>Installation</h1>
<pre><code>pip install MathLibrary</code></pre>

<h1>Example</h1>
<pre>
<code class="python">
        
        from MathLibrary.mathlibrary import MathLibrary
        
        MathLibrary.fact(5)
        
        
<code>
</pre>
<h5>Output : </h5>
<pre><code>120</code></pre>
